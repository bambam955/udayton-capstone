# BizRush

A platform for small businesses to rush-deliver supplies quickly and accurately.

## Getting Started

1. Install [`just`](https://github.com/casey/just?tab=readme-ov-file#installation).
2. Install Flutter and the Android SDK.
    - Directions [here](./CONTRIBUTING.md#flutter--android-sdk-setup)
3. To run the app in your browser:

    ```bash
    just run-web
    ```

4. To run the app in an Android emulator:

    ```bash
    just run-android
    ```
