# BizRush

A platform for small businesses to rush-deliver supplies quickly and accurately.

## Getting Started

1. Install [Docker](https://docs.docker.com/get-docker/) and [`just`](https://github.com/casey/just?tab=readme-ov-file#installation).
2. One-time setup:

    ```bash
    just setup
    ```

3. To run the main BizRush app in your browser:

    ```bash
    just up main-web
    ```

4. To run the main BizRush app in an Android emulator (see [CONTRIBUTING.md](./CONTRIBUTING.md#running-on-android) for emulator setup):

    ```bash
    just up main-android
    ```
